🎉 Welcome to Data Scraper!

Your powerful Chrome extension for extracting data from web pages and sending it to webhooks

🚀 What You Can Do

This extension allows you to:

🔍

Scrape Data

Extract text, links, images, and more from any webpage using CSS selectors

📡

Send to Webhook

Automatically send scraped data to your webhook endpoint

⚙️

Custom Rules

Define your own scraping rules using JSON configuration

📊

Track History

Keep track of all your scraping activities and results

📋 Quick Start Guide

Step 1: Configure Your Webhook

Click the extension icon in your toolbar and enter your webhook URL where you want to receive the scraped data.

Step 2: Set Up Scraping Rules

Define what data you want to extract using CSS selectors. Here's an example:

{ "title": "h1", "content": ".content, .main, .body", "links": "a[href]", "images": "img[src]", "price": ".price, [data-price]", "rating": ".rating, [data-rating]" }

Step 3: Start Scraping

Navigate to any webpage, click the extension icon, and hit "Scrape Current Page" to extract data and send it to your webhook.

💡 Pro Tips

  • Use the browser's developer tools to inspect elements and find the right CSS selectors
  • Test your selectors on different pages to ensure they work consistently
  • Use multiple selectors separated by commas for fallback options
  • Right-click on any page to access quick scraping options
  • Check the extension's history to review past scraping results

🔧 Advanced Features

The extension also supports:

  • Context Menu Integration: Right-click to scrape selected text or entire pages
  • Keyboard Shortcuts: Use Ctrl+Shift+S (or Cmd+Shift+S on Mac) to quickly open the scraper
  • Automatic Cleanup: Old scraping history is automatically cleaned up after a week
  • Error Handling: Comprehensive error reporting and retry mechanisms
